AstroMenace для Linux

Версия 1.3.2

 
Практически все популярные дистрибутивы Linux уже имеют в своих репозиториях стабильную версию игры. Установите пакет "astromenace" с помощью пакетного менеджера вашего дистрибутива. Если пакет с игрой не найден, свяжитесь с мейнтейнерами вашего дистрибутива Linux.
 
 

Установка.

 
  • Ubuntu/Mint/Debian:
    $ sudo apt-get install astromenace
     
  • Mageia:
    $ sudo urpmi astromenace
     
  • OpenSuse:
    $ sudo zypper install astromenace
     
  • Fedora:
    $ sudo yum install astromenace
     
  • FreeBSD:
    $ sudo pkg_add -r -v astromenace
     
 

Сборка из исходного кода.

 
  • Убедитесь, что все необходимые пакеты с хидерами для компиляции игры установлены: gcc (with g++), cmake, libsdl, libogg, libvorbis, GLU, openal, alut (freealut), libXinerama, freetype2
  • Ubuntu/Mint/Debian:
    $ sudo apt-get install g++ cmake make libsdl1.2-dev \
    libogg-dev libvorbis-dev libopenal-dev libalut-dev libXinerama-dev \
    libfreetype6-dev libfontconfig1-dev fonts-linuxlibertine
  • Mageia:
    $ sudo urpmi gcc-c++ cmake make libSDL-devel libogg-devel \
    libvorbis-devel libopenal-devel libxinerama-devel libfreealut-devel \
    freetype2-devel libfontconfig-devel fonts-ttf-libertine
  • OpenSuse:
    $ sudo zypper install gcc-c++ cmake make libSDL-devel \
    libogg-devel libvorbis-devel openal-soft-devel libXinerama-devel freealut-devel \
    freetype2-devel fontconfig-devel linux-libertine-fonts
  • Fedora:
    $ sudo yum install gcc-c++ cmake make SDL-devel libogg-devel \
    libvorbis-devel openal-soft-devel libXinerama-devel freealut-devel \
    freetype-devel fontconfig-devel linux-libertine-fonts linux-libertine-biolinum-fonts
  • FreeBSD:
    $ sudo pkg_add -r -v gcc cmake sdl libogg libvorbis openal \
    libXinerama freealut freetype2 fontconfig linuxlibertine
  • Cкачайте последний стабильный релиз

    $ cd ~/
    $ wget https://github.com/viewizard/astromenace/archive/v1.3.2.tar.gz
    $ mkdir -p ./AstroMenace
    $ tar -xzvf v1.3.2.tar.gz -C ./AstroMenace --strip-components=1
  • Скомпилируйте бинарный файл игры

    $ cd ~/AstroMenace
    $ cmake ./
    $ make
  • Создайте VFS файл данных игры

    $ ./AstroMenace --pack --rawdata=./RAW_VFS_DATA
  • Создайте ярлык на рабочем столе или запустите AstroMenace

    $ ~/AstroMenace/AstroMenace
